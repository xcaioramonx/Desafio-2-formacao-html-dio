# Desafio-2-formacao-html-dio
Projeto de site de uma clínica médica desenvolvido para o desafio da DIO, criando um site completo usando HTML e complementando com alguns recursos de CSS aprendidos anteriormente em outro curso. Inclui páginas interligadas, tabela de horários e navegação estruturada. As imagens foram selecionadas no Unsplash. Layout simples, limpo e funcional.
